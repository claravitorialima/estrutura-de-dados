# estrutura-de-dados
# 📄 Atividade: Lista de alunos em JavaScript

Este repositório tem um exercício prático de JavaScript focado na criação de estruturas de dados utilizando **Arrays** e 5 **Objetos**, além do acesso a propriedades específicas.
---

## 📋 Requisitos do Desafio

1. Criar um array chamado `alunos` contendo **5 objetos**.
2. Cada objeto (aluno) deve possuir as seguintes propriedades:
   * `nome`
   * `idade`
   * `disciplinaPreferida`
3. Acessar e exibir:
   * O nome do **primeiro** aluno.
   * A disciplina preferida do **último** aluno.
